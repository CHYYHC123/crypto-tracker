# Changelog

## 1.0.12 - 2025-01-XX

- ✨ 新增价格预警功能：支持设置价格上限/下限预警，价格触发时显示 Toast 通知
- ✨ 新增多数据源支持：支持切换 OKX、Gate、BN、HL 等多个交易所数据源
- ✨ 新增币种拖拽排序：在悬浮组件中支持拖拽调整币种显示顺序
- ✨ 新增网络状态显示：实时显示数据连接状态（LIVE/DEGRADED/OFFLINE）
- 🎨 新增移动端适配：移动端自动隐藏悬浮组件，优化移动端体验
- 🔧 优化 WebSocket 假死检测：自动检测并恢复假死的 WebSocket 连接
- 🔧 优化内容同步机制：页面可见时自动同步最新数据
- 🔧 优化币种顺序更新：仅顺序变化时不触发 WebSocket 重连，提升性能
